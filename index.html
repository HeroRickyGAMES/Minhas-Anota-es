Política de Privacidade

O Aplicativo Minhas Anotações usa banco de dados externo para registrar os dados digitados no aplicativo!

O aplicativo também registra o seu email e seu username no banco de dados!

Não coletamos nenhum dado que possa comprometer o usuario!

E nem usamos de forma comercial!

O Aplicativo precisa de acesso a internet para fazer as anotações!
